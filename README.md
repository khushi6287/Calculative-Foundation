# Calculative Foundation Project

A comprehensive Python project demonstrating fundamental concepts in linear algebra, vector mathematics, matrix operations, and dimensionality reduction techniques using educational dataset of student scores.

## 📋 Overview

This project provides hands-on implementations of core mathematical concepts used in data science and machine learning, including vector operations, matrix manipulations, eigenvalue decomposition, and dimensionality reduction techniques. The analysis is performed on a synthetic dataset of student scores across four subjects.

## 🎯 Features

### Dataset
- 5 students × 4 subjects (Math, Physics, Chemistry, English)
- Synthetic academic performance data for demonstration purposes

### Vector Mathematics
- Vector representation of student data
- L1 and L2 norm calculations
- Dot product and angle between vectors
- Cross product computation
- Vector projections

### Matrix Operations
- Matrix addition and multiplication
- Transpose operations
- Determinant calculation
- Matrix inversion
- Covariance matrix computation

### Decomposition Methods
- **Eigen decomposition** of covariance matrix
- **LU decomposition** for matrix factorization
- **Singular Value Decomposition (SVD)** for comprehensive matrix analysis

### Dimensionality Reduction
- **Principal Component Analysis (PCA)** for 2D visualization
- **Linear Discriminant Analysis (LDA)** for classification tasks

## 📊 Visualizations

The project includes several visual representations:
- Histogram of math score distribution
- Bar chart of average scores per subject
- Correlation matrix heatmap
- 2D vector visualization
- PCA projection scatter plot
